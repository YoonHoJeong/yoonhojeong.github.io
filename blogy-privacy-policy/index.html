블로기(Blogy) 크롬 확장 프로그램 개인정보처리방침

요약: 블로기는 개인정보를 수집하거나 저장하지 않습니다. 모든 데이터는 기능 제공 목적으로만 임시 처리되며 즉시 삭제됩니다.

1. 개인정보 수집 현황

블로기 확장 프로그램은 개인정보를 수집하지 않습니다.

다음과 같은 정보는 수집하지 않습니다:

  • 이름, 이메일 주소, 전화번호 등 개인 식별 정보
  • 로그인 정보 또는 계정 데이터
  • 브라우징 기록
  • 위치 정보
  • 쿠키나 기타 추적 데이터

2. 처리하는 데이터 유형

확장 프로그램이 기능 제공을 위해 임시로 처리하는 데이터:

  • 블로그 컨텐츠: 네이버 블로그 에디터에서 추출한 텍스트, 이미지 URL
  • 키워드: 사용자가 입력한 검색 키워드
  • 분석 결과: 키워드 분석 및 AI 처리 결과

중요: 이러한 데이터는 기능 실행 중에만 메모리에서 처리되며, 확장 프로그램이나 외부 서버에 저장되지 않습니다.

3. 데이터 사용 목적

처리되는 데이터의 사용 목적:

  • 컨텐츠 추출: 네이버 블로그 작성 내용을 구조화하여 표시
  • AI 개선 제안: Claude AI를 통한 글 품질 향상 제안
  • 키워드 분석: 네이버 API를 통한 SEO 분석 정보 제공
  • 이미지 다운로드: 블로그 이미지의 로컬 저장 기능 제공

4. 외부 서비스 연동

기능 제공을 위해 다음 외부 서비스와 통신합니다:

4.1 Claude AI (Anthropic)

  • 전송 데이터: 블로그 텍스트 컨텐츠
  • 목적: 컨텐츠 품질 개선 제안
  • 보관 기간: 즉시 삭제 (세션 종료 시)

4.2 네이버 공식 API

  • 전송 데이터: 검색 키워드
  • 목적: 블로그 발행량, 검색 트렌드, 연관 키워드 분석
  • API 종류: 네이버 검색 API, 데이터랩 API, 검색광고 API

5. 데이터 보안 및 보관

  • 로컬 처리: 모든 데이터는 사용자의 브라우저에서만 처리
  • 즉시 삭제: 기능 실행 완료 후 메모리에서 즉시 제거
  • 암호화 통신: 모든 외부 API 통신은 HTTPS 사용
  • 저장 금지: 확장 프로그램은 어떠한 사용자 데이터도 저장하지 않음

6. 사용자 권리

개인정보를 수집하지 않으므로 별도의 삭제 요청 절차는 없습니다. 사용자는 언제든지:

  • 확장 프로그램 사용을 중단할 수 있습니다
  • 확장 프로그램을 제거할 수 있습니다
  • 특정 권한을 비활성화할 수 있습니다

7. 권한 사용 내역

확장 프로그램이 요청하는 권한과 용도:

  • activeTab: 현재 탭이 네이버 블로그인지 확인
  • scripting: 블로그 컨텐츠 추출을 위한 스크립트 실행
  • 네이버 도메인: 블로그 컨텐츠 접근 및 API 통신
  • pstatic.net: 네이버 블로그 이미지 다운로드

8. 쿠키 및 추적

블로기는 다음을 사용하지 않습니다:

  • 쿠키 (Cookies)
  • 로컬 스토리지
  • 사용자 추적 기술
  • 분석 도구 (Google Analytics 등)

9. 정책 변경

본 개인정보처리방침이 변경되는 경우:

  • Chrome Web Store를 통해 업데이트 공지
  • 중요한 변경사항은 확장 프로그램 내에서 알림
  • 변경된 정책은 즉시 적용

10. 법적 근거

데이터 처리의 법적 근거:

  • 사용자 동의: 확장 프로그램 설치 및 사용
  • 정당한 이익: 사용자가 요청한 기능 제공
  • 계약 이행: 블로그 작성 도구 서비스 제공
